Generating "EvalGPT"

05 May 2023 on Openai
Note: This post provides an overview of the experimental EvalGPT project.
<EvalGPT. Is a Ruby based OpenAI client designed to write and run code you generate from prompts.

The project was generated interactively using chatgpt4, and currently supports generating and running ruby and swift with experimental support for more languages.
